# PixlCore.com Website Repo

This repository holds all the files for the `PixlCore.com` website.

It is designed as a [PoolNoodle App](https://github.com/jhuckaby/PoolNoodle#app-installer) and can be installed using the `noodle` CLI:

```sh
noodle install github:pixlcore/www.pixlcore.com#v1.0.0
```
